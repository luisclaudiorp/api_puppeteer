## Description

Service developed to capture information from websites using nestjs and puppeteer

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ Default port is set to 3000

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
